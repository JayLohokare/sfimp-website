# SF Indian Music Project Website

Official public-facing website for the San Francisco Indian Music Project.

## Vision

Building the largest Asian music community in the Bay Area through:
- **Music Excellence** - Technical mastery meets soul and emotion
- **Community First** - Decentralized community built to outlast any single person
- **Sustainable Growth** - Non-profit model with revenue reinvestment

## Features

- Clean, sleek modern design
- Instagram integration
- Mobile responsive
- Fast loading performance
- GitHub Pages hosting

## Deployment

This site is hosted on GitHub Pages. To deploy:

1. Make sure you're in the `sfimp-website` directory
2. Commit your changes
3. Push to the `gh-pages` branch

```bash
cd sfimp-website
git add .
git commit -m "Update website"
git push origin gh-pages
```

Or use the deploy script:
```bash
./deploy.sh
```
